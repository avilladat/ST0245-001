El programa es un menu que le permite interactuar con distintos
tipos de datos abstractos, como puntos, fechas y lineas.

Para inicializar el programa ejecute la clase "Ejecutar", al hacerlo
se mostrara un peque√±o menu, en el cual podra seleccionar el tipo de 
dato abstracto con el que desea interactuar.

Dependiendo de la opcion que elija se le pedira la informacion correspondiente,
y se desplegara otro menu con las opciones que puede realizar.

El resto de la funcionalidad es bastante intuitiva.
